# TURNERO BANKING 🏧
APP DE TURNOS PARA ASISTIR AL BANCO.
# Trabajo realizado
Una app desarrollada con Angular para resolver problema de capacidad en los bancos debido a la situacion de COVID-19.
# Comenzando  🚀
```
# Instalar dependencias para el cliente
npm install

# Correr la app
ng serve -o
```
# Construido con 🛠️
- Angular 
- Typescript
- NGRX
- RXJS
- SASS
- PWA
